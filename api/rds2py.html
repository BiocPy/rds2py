rds2py.interface.as_summarized_experiment(robj)[source]¶

Parse an R object as a SingleCellExperiment or SummarizedExperiment.

Note: This function demonstrates how to parse a complex RDS objects in Python and may not work for all scenarios.

Parameters:

  • robj –

    Object parsed from the RDS file.

    Usually the result of read_rds().

  • order –

    Row-major (C-style) or Column-major (**F**ortran-style) order. Only used if the robj contains a ndarray.

    Defaults to “F”.

Returns:

A SummarizedExperiment or SingleCellExperiment from the R object.
