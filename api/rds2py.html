Read an R object as a ndarray.

  • robj (MutableMapping) – Object parsed from the RDS file. Usually the result of read_rds().

  • order (Literal["C", "F"]) – Row-major (C-style) or Column-major (**F**ortran-style) order. Defaults to “F”.

rds2py.interface.as_summarized_experiment(robj: MutableMapping) → SummarizedExperiment | SingleCellExperiment[source]#

Read an R object as a SingleCellExperiment or SummarizedExperiment.

Note: This function demonstrates how to parse a complex RDS object.

Parameters:

  • robj (MutableMapping) – Object parsed from the RDS file. Usually the result of read_rds().

  • order (Literal["C", "F"]) –

    Row-major (C-style) or Column-major (**F**ortran-style) order.

    Only used if the robj contains a ndarray.

    Defaults to “F”.

Raises:

TypeError – If robj is not a supported class.

Returns:

A SummarizedExperiment or SingleCellExperiment from the R object.

Return type:

Union[SummarizedExperiment, SingleCellExperiment]
